## - Descripción del Proyecto

Este repositorio contiene la aplicación **ElectroFactura**, desarrollada para cumplir con los requisitos de la asignatura de **Desarrollo de Interfaces**. Aquí encontrarás todo lo necesario para su descarga e instalación.

---

## - Requisitos del Sistema

Antes de comenzar con la instalación, asegúrate de que tu equipo cumple con los siguientes requisitos mínimos:

-   - **Java Runtime Environment (JRE):** Versión **21** o **23**.
-   - **Memoria RAM:** 400 MB disponibles como mínimo.
-   - **Sistema Operativo:** Compatible con **Windows**, **Linux** o **macOS**.

---

## - Instalación del Programa

Sigue estos sencillos pasos para poner en marcha ElectroFactura:

1.  **Verifica los requisitos:** Asegúrate de tener la versión correcta de Java instalada.
2.  **Descarga y descomprime:** Extrae el contenido del archivo ElectroFactura.zip en la carpeta que prefieras.
3.  **Ejecuta la aplicación:** Navega hasta la carpeta descomprimida y haz doble clic sobre el archivo **ElectroFacturaApp.exe** (en Windows) o el ejecutable correspondiente para tu sistema operativo.
